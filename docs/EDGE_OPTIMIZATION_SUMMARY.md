### 2. ✅ 快速部署配置

**文件**：`config-edge-rdk.yaml`

**特点**：
- 专门针对 RDK 硬件优化
- 内存受限参数预设（4GB 预算）
- Qwen-1.5B GGML 后端配置
- 禁用并发，严格单线程
- 包含详细注释和调优建议

**关键参数**：
```yaml
llm:
  engine: "ggml"  # 改用 GGML 而非 vLLM
  model_path: "models/llm/qwen-1.5b-chat-q3_k_m.gguf"
  n_ctx: 512  # 严格限制上下文
  n_gpu_layers: -1  # 启用 NPU
  
pipeline:
  max_concurrent_tasks: 1  # 禁止并发
  sequential_only: true
```
